我的饮食计划
高碳日（训练日）

利用提高的胰岛素敏感性，将能量优先填入肌肉糖原而非脂肪细胞，同时“欺骗”身体，避免长期低碳导致的新陈代谢适应性下降，并满足训练能量需求。

热量缺口10-20%。 建议比例：3g碳水 / 2g蛋白质 / 0.8g脂肪 (每公斤体重)

  • 早餐： 全天30%的碳水 + 30g的蛋白质 + 适量的脂肪。
  • 午餐： 不吃碳水，只摄入蛋白质和脂肪。
  • 训前加餐（训练前2-3小时）： 全天20%的碳水。
  • 训后餐/晚餐： 全天50%的高分解速度碳水 + 30-50g蛋白质 + 20g低脂。
低碳日（休息日）

目的：让身体进入周期性营养性酮症，身体察觉没有碳水后，会进行糖异生，也就是用蛋白质转换葡萄糖。均摊碳水，稳定胰岛素水平。

建议比例：1.5g碳水 / 2g蛋白质 / 剩余热量由脂肪补充 (每公斤体重)

  • 早餐、午餐、晚餐： 蛋白质、碳水和脂肪平均分配到三餐。
  • 有氧训练后： 喝两勺蛋白粉用特仑苏牛奶冲服。
